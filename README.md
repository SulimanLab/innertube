# innertube
Python Client for Google's InnerTube API (works with YouTube, YouTube Music etc.)

### Clients
|         | YouTube | YouTube Music | YouTube Kids |
| ------- | ------- | ------------- | ------------ |
| Web     | WEB     | WEB_REMIX     | tbc.         |
| Android | ANDROID | ANDROID_MUSIC | tbc.         |
| Ios     | IOS     | IOS_MUSIC     | tbc.         |
